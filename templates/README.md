# templates

이 폴더는 향후 `SECURITY_REPORT.md`, `SECURITY_TODO.md` 같은 출력물 템플릿과 예시 파일을 둘 위치입니다.

v0.1에서는 템플릿 엔진을 따로 두지 않고 `packages/core`에서 Markdown을 직접 생성합니다. 아직 템플릿을 분리하지 않은 이유는 초기 규칙과 문서 구조가 자주 바뀔 수 있기 때문입니다.

> English: This directory is reserved for future Markdown templates and output examples. v0.1 generates Markdown directly from `packages/core`.

## 앞으로 담을 수 있는 것

- 내부 공유용 보안 점검 보고서 템플릿
- 개발자 TODO 체크리스트 템플릿
- 한국어/영어 보고서 예시
- 팀별 커스터마이징 가이드

## 현재 상태

- 템플릿 파일 없음
- 런타임 템플릿 로딩 없음
- MCP 서버나 외부 도구 연동 없음
